Project created by Santiago Rivera as a personal project.

What to know:
- This project handles commands inputted by users on discord chat whenever summoned.
- The features of this robot include being able to: check user's balance, beg to gain balance, deletes it's previous messages, interact witrh messages to assign users roles, display a "How to menu", basic ping pong bot.
- Used in this project: javacript to handle commands, MongoDB to store user's currency info, Discord.js library to use Discord features/info.
- Token that makes this bot unique is random on here and serves less of a security risk, real token is in my possession
